Project Overview

The Carrom Board Game is a digital adaptation of the popular tabletop game, implemented using C++ and the SDL library. It provides a realistic physics-based simulation of the classic game, allowing players to enjoy Carrom on their computers with intuitive controls and engaging gameplay.

Key Features

  • Realistic physics simulation using Box2D physics engine
  • Multiple game modes: Player vs Player, Player vs AI
  • Customizable striker force and angle
  • Realistic collision detection and response
  • Scoring system that follows official Carrom rules
  • Sound effects and background music
  • Save and resume game functionality

Technical Challenges

One of the main challenges in developing this game was implementing accurate physics for the pieces. The movement of the striker and the resulting collisions needed to feel realistic while remaining performant. This required fine-tuning of the physics parameters and extensive testing.

Another significant challenge was creating an AI opponent that could make strategic shots. The AI algorithm analyzes the board state, calculates possible trajectories, and selects the most promising shot based on a scoring system that evaluates potential outcomes.

Technologies Used

C++ SDL2 Box2D Physics OpenGL CMake

Development Process

This project was developed over a period of 3 months, starting with basic rendering and gradually adding physics, game logic, AI, and polish. The development followed these main phases:

  1. Research and planning phase
  2. Setting up the project structure and rendering engine
  3. Implementing core physics for pieces and striker
  4. Building game logic and rules
  5. Creating AI opponent
  6. Adding UI, sound effects, and visual polish
  7. Testing and optimization

What I Learned

This project significantly enhanced my understanding of game physics, collision detection, and AI development. I learned to balance realistic physics with enjoyable gameplay, and gained experience in performance optimization for real-time applications.

The AI implementation taught me about decision trees and evaluation functions, while the project as a whole improved my C++ skills and understanding of game development patterns.
